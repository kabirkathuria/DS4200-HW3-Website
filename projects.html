MovieLens Movie Recommendation System

Created a Movie Recommendation System Engine for my capstone project in the HarvardX: Data Science Professional Certificate Program. In this project, I refined a machine learning algorithm for a recommendation system tasked to predict movie ratings based on user preferences. These are some key highlights from the project:

  • Wrangled and tidied a 10M MovieLens dataset to build a movie recommendation engine that predicts trends in user movie ratings.
  • Trained mean movie rating, movie effect, and user impact ML models and evaluated their accuracy using a validation set.
  • Enhanced recommendation performance based on user preferences, minimizing the Root Mean Squared Error score to 0.864.
